锯木头问题

运算法则：
1. 将一根木头锯成n段需要(n-1)次切割
2. 总时间 = 切割次数 × 每次耗时
3. 多人同时操作可以并行优化时间
将木头锯成5段，每次锯切需要2分钟，总共需要多少时间？
(5-1)×2 = 8分钟
锯切一根木头7次，可以得到多少段木头？
7+1 = 8段
工人用3分钟锯一次，要得到10段木头，最少需要多少分钟？
(10-1)×3 = 27分钟

爬楼梯问题

运算法则：
1. 爬的层数 = 终点楼层 - 起点楼层
2. 总时间 = 爬的层数 × 每层耗时
3. 下楼时计算方法相同
从1楼到6楼，每层需要30秒，总共需要多少时间？
(6-1)×30 = 150秒 = 2分30秒
从3楼到9楼，每层耗时45秒，需要多少时间？
(9-3)×45 = 270秒 = 4分30秒
从8楼下到1楼，每层需要20秒，总共需要多少时间？
(8-1)×20 = 140秒 = 2分20秒

运算法则：
1. 确定看错的位数（个位、十位等）
2. 计算看错数字与原数字的差值
3. 根据运算类型（加、减、乘、除）调整结果

5. 十位看错：差值 = (看错数字 - 原数字)×10

多了(8-3)=5；正确和=错误和-5

多了(8-5)×10=30
小刚把因数个位上的9看成了6，积会减少多少？
减少=(9-6)×另一个因数

1. 可以同时进行的任务，总时间取最长任务时间

max(15,5)=15分钟

排队问题

运算法则：
1. 队伍总人数 = 前面人数 + 自己 + 后面人数
2. 两人中间人数 = 位置差 - 1
3. 从前面数第n个 = 位置n
4. 从后面数第m个 = 总人数 - m + 1
小丽前面有5人，后面有7人，队伍共有多少人？
5+1+7=13人
小明排第8，他前面有多少人？
8-1=7人
小红从前面数是第4，从后面数是第6，队伍共有多少人？
4+6-1=9人

统计图表问题

运算法则：
1. 比较柱状图高度或饼图比例
2. 计算各部分之间的差值或倍数关系
3. 总数 = 各部分数量之和
4. 比例 = 部分数量 / 总数
图表显示A班30人，B班是A班的1.5倍，B班有多少人？
30×1.5=45人
三班共120人，图表显示C班占25%，C班有多少人？
120×25%=30人
图表中最高柱代表50，最低柱代表20，相差多少？
50-20=30

天平平衡问题

运算法则：
1. 天平平衡时：左边重量 = 右边重量
2. 不平衡时：较重的一边下沉
3. 利用已知重量推导未知重量
4. 比例关系：A/B = C/D
天平左边2个苹果，右边6个草莓，已知1个苹果=3个草莓，天平会怎样？
左边=2×3=6个草莓重量=右边，天平平衡
1个西瓜等于4个菠萝，1个菠萝等于3个苹果，1个西瓜等于多少个苹果？
4×3=12个苹果
天平左边3个方块，右边9个球，方块和球的重量比是多少？
1个方块=3个球
